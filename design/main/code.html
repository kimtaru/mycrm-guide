AI 네이티브 DX

기존의 UI 라이브러리는 중첩된 복잡성 때문에 LLM이 사용하기 어렵습니다. mycrm은 **AI와의 협업**을 염두에 두고 구축되었습니다. 구조화된 스키마 정의와 시맨틱 컴포넌트 마커로, LLM·코딩 에이전트가 읽고 확장하기 쉬운 형태를 지향합니다.

  • check_circle 높은 LLM 정확도를 위한 시맨틱 HTML 구조
  • check_circle 프롬프트 엔지니어링에 최적화된 .md 참조 파일
  • check_circle 상용구 코드를 제거하는 Prop 기반 로직
