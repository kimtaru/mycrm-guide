Getting Started

Welcome to mycrm UI, a collection of highly-performant, architecturally-sound components designed for data-rich applications.

download

Installation

Install the core library using your preferred package manager. Ensure you have React 18+ and Tailwind CSS configured in your project.

npm 
npm install mycrm
yarn 
yarn add mycrm
play_circle

Basic Usage

1

Setup the Provider

Wrap your application with the MyCRMProvider to enable global theming and context. 

import { MyCRMProvider } from 'mycrm';

function App() {
  return (
    <MyCRMProvider theme="light">
      <Dashboard />
    </MyCRMProvider>
  );
}
2

Import and Use Components

Now you can start using any component. Here's a simple example of the Button component. 

import { Button } from 'mycrm';

const MyPage = () => (
  <Button variant="primary" size="lg">
    Create Project
  </Button>
);
